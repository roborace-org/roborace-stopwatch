# Roborace Stopwatch

Секундомер для измерения времени за которое автономный робот проходит круг.

## Запуск проекта с помощью [PlatformIO](http://platformio.org/)

### Сборка 
```bash
pio run
```

### Загрузка на плату
```bash
pio run -t upload
```

### Настройка среды Clion
```bash
pio init --ide clion --board nanoatmega328
```
